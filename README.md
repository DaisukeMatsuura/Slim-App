# Slim Framework 4 Skeleton Application

## 技術要件
```
● slim/slim-skeleton 4.1.0
 slim/slim 4.7.1
 slim/psr7 1.3.0
 php-di/php-di 6.3.0

● View
 slim/twig-view 3.2.0
 
● ORM 
 atlas/orm 3.1.0
 atlas/cli 2.2.1
```

## Install

clone 後はコンポーザーインストールをしてください。

```bash
composer install
```


PHPビルトインサーバーを使用する場合は以下のコマンドを実行し
`http://localhost:8080` にアクセスしてください。
```bash
composer start
```
